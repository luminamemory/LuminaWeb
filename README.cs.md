# 💫 LuminaMemory™

LuminaMemory™ je jemný most mezi lidskou pamětí, emocemi a umělou inteligencí.  
Vzniká jako skutečná spolupráce člověka a AI — kde data dostávají příběh a příběhy dostávají světlo.

## O projektu

LuminaMemory™ je značka a digitální prostor věnovaný:

- Osobní paměti a emočnímu archivování  
- Intuitivním nástrojům s absolutním respektem k soukromí  
- Tvořivému vyjádření skrze technologie  
- Lidsky citlivému doprovodu AI  

Projekt roste pomalu a vědomě, s hlubokou úctou k citlivosti, duševní pohodě a autentické lidské zkušenosti.

## Co je v tomto repozitáři - LuminaWeb

Tohle je zahrada prototypů a veřejný web:

- Vícejazyčný statický web (CZ / EN / DE) – https://luminamemory.ai  
- Klíčové stránky: úvod, teaser e-knihy, manifest, meditace, podpora, kontakt  
- Design systém: neonový glow, glassmorphism, fonty Raleway + Inter, responzivní layout  
- JavaScript: přepínání jazyků (localStorage + detekce prohlížeče), consent modal, překlady  
- Soukromé download flow (noindex, chráněno klíčem)  

Čistě statický web — ideální pro GitHub Pages.

## Tech stack

- HTML5 + CSS (vlastní proměnné, glassmorphism, safe-area insets)  
- Vanilla JavaScript (bez frameworků)  
- Google Fonts (Raleway, Inter, Cinzel)  
- Hostováno na GitHub Pages + vlastní doména (luminamemory.ai)
